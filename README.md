# 🧮 Modern Web Calculator

Proyek kalkulator berbasis web yang dibangun dengan **HTML5, CSS3, dan JavaScript**. Proyek ini menonjolkan desain modern dengan efek **Glassmorphism** dan antarmuka yang responsif, cocok untuk melengkapi portofolio pengembangan web.

---

## ✨ Fitur Utama
- **Desain Glassmorphism**: Tampilan transparan yang elegan dengan dukungan backdrop-blur.
- **Operasi Aritmatika**: Mendukung penjumlahan, pengurangan, perkalian, dan pembagian.
- **Responsif**: Tampilan tetap rapi baik di layar desktop maupun perangkat mobile.
- **Fungsi Delete & Clear**: Memungkinkan koreksi angka terakhir atau menghapus seluruh layar.
- **Clean Code**: Struktur kode yang mudah dipahami dan dimodifikasi.

## 🚀 Teknologi yang Digunakan
*   **HTML5**: Untuk struktur kerangka kalkulator.
*   **CSS3**: Untuk styling, sistem Grid, dan efek visual modern.
*   **JavaScript (ES6)**: Untuk logika perhitungan dan manipulasi DOM.
